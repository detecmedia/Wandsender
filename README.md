# Wandsender 1- / 3- fach - Homematic 868Mhz
## benötigte Hardware
 * 1x Funk Wandsender Licht Wandschalter 433MHZ für RF Empfänger Fernbedienung [Ebay](https://www.ebay.de/itm/Funkschalter-System-230V-WIFI-RF-433Mhz-Funk-LED-Modul-Licht-Taster-Schalter/283197145432?hash=item41efdd8158:m:mX2GEfhY_EjQCHIkcy7EHgQ) oder [Ebay](https://www.ebay.de/itm/Funk-Wand-Lichtschalter-Wandschalter-433MHZ-fur-Empfanger-Fernbedienung-HH/274152870519?hash=item3fd4c8d277:m:mR3WXSzJe0iaSnNkx7MufLw) oder [aliexpress](https://www.aliexpress.com/item/4000307038181.html?spm=a2g0o.productlist.0.0.723f76bbhxU5Mg&algo_pvid=5d0559ee-5ff2-4da6-85b1-1a6fa0cb1dc0&algo_expid=5d0559ee-5ff2-4da6-85b1-1a6fa0cb1dc0-9&btsid=0bb47a1915816099683175964ed1eb&ws_ab_test=searchweb0_0,searchweb201602_,searchweb201603_)
 * 1x Arduino Pro Mini **ATmega328P (3.3V/8MHz)**
 * 1x CC1101 Funkmodul **(868 MHz)**
 * 1x FTDI Adapter (wird nur zum Flashen benötigt)
 * 1x Taster 6*6
 * 1x Batteriehalter print für CR2032 **Reichelt KZH 20P** oder [EBAY](https://www.ebay.de/itm/CR2012-2032-Knopfzellenhalter-max-20-mm-Silber-Printmontage-horizontal-2/253806533551?hash=item3b180c5baf:g:IOUAAOSwtlZb0KwD:rk:2:pf:0)
 * 1x Batterie CR2032

Optional die LEDS **Verringert die Batterielaufzeit**
* 2x Widerstand 330 Ohm
* 2x LED 3mm
